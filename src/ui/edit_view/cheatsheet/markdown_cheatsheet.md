# Markdown Syntax

## Headings

##### Heading 1
`# Heading`
H1

##### Heading 2
`## Heading`
H2

##### Heading 3
`### Heading`
H3

##### Heading 4
`#### Heading`
H4

##### Heading 5
`##### Heading`
H5

##### Heading 6
`###### Heading`
H6

## Emphasis

`**text**`
`__text__`
Displays text in **bold**

`*text*`
`_text_`
Displays text in _italics_

`~~text~~`
Adds ~~strikethrough~~ effect to the text

`==text==`
Adds ==highlight== effect to the text

## Lists

##### Unordered Lists
`- list item`
`- list item`

##### Ordered Lists
`1. list item`
`2. list item`

## Links

##### Link
`[link label](https://linkurl.io)`

##### Image
`![Image alt text](https://imageurl.io)`

## Blockquotes

`> quote`

## Horizontal Rule

`- - -`

## Code

##### Code span
`code`

##### Code block
```language-name
code
```
